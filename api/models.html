class chexus.BucketItem(file_path, file_name=None, checksum=None, key=None)[source]¶

Represents an object in an AWS S3 bucket

Parameters

  • file_path (str) – The path to the file.

  • file_name (str) – The Name of the file. This attribute is set according to the file’s path if a name is not provided.

  • checksum (str) – The checksum of the file. This attribute is set if a checksum is not provided and the file exists.

  • key (str) – The object key of the S3 file object. This attribute is set with the name attribute if no key is provided.
